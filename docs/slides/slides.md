- Search pokemons
- Display list of 'found' pokemons



# Setup
Make sure you have nodejs installed (v7.5.0)
```shell
mkdir pokedex
cd pokedex
npm i -g create-react-app
create-react-app pokedex
cd pokedex
npm start
```
